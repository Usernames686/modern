以下目录是本地安装或构建产物，仓库已通过 `.gitignore` 排除。若人工压缩文件夹交付，建议先删除或忽略：

## 6. 交付建议

- Git 交付：直接提交源码、文档、SQL、上传资源，保持 `.gitignore` 生效。
- 压缩包交付：先删除 `node_modules`、`dist`、`target`、日志和系统垃圾文件，再压缩 `modern/`。
- 生产启用真实支付、退款、短信、公众号发布、物流同步前，必须单独配置生产参数并专项验收。
