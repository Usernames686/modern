- 这些文件是原始资料副本，不代表新版 `modern` 的所有实现细节都逐字相同。
- 新版运行和部署优先看 `../交付说明.md`、`../开发运维手册.md` 和根目录 `README.md`。
- 接口路径仍保持 `/api/admin/**`、`/api/front/**`、`/api/public/**` 兼容。
- 数据库结构继续使用老 `single_open` 和 `eb_*` 表，当前可运行快照在 `../../legacy-deps/db/single_open_current.sql`。
