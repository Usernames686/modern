# 《CRMEB Modern》系统测试说明书

## 1. 测试目标

验证 CRMEB Modern 能作为独立项目交付，重点检查源码、数据库、素材、Docker、后台、H5、接口和第三方安全边界。

## 2. 自动化测试

### 2.1 全量审计

```bash
cd modern
sh scripts/run-delivery-audit.sh full
```

### 2.2 后端

```bash
cd modern/backend
sh scripts/check-delivery.sh
```

### 2.3 后台

```bash
cd modern/admin-web
npm run check:delivery
npm run build
```

### 2.4 H5

```bash
cd modern/app-web
npm run check:delivery
npm run build
```

## 5. 数据一致性测试

- 后台新增商品，H5 能看到。
- 后台修改价格，H5 商品详情同步。
- H5 下单，后台订单列表同步。
- 后台处理退款，H5 售后状态同步。
- 后台保存装修，H5 首页/分类页同步。

## 6. 安全测试

- 无 token 访问后台保护接口返回 401。
- 不提交 `.env`、证书和私钥。
- 微信支付、支付宝支付、真实退款、短信、公众号发布、物流同步默认不误触发。
